Task 1. LongEval-Retrieval

The data for this task is a sequence of web document collections and queries provided by Qwant. Each document set will have a release time stamp, with the first set (in chronological order) being the training data.


Task 2. LongEval-Classification

For model training, we plan to use the TM-Senti dataset and extend it with a novel test set for evaluation of participants's' submissions. TM-Senti is a general large-scale tweets sentiment dataset in the English language spanning a 9-year period ranging from 2013 to 2021. Tweets are labelled for sentiment as either “positive” or “negative”. The annotation is performed using distant supervision based on a manually curated list of emojis and emoticons and, thus, can be easily extended to cover more recent years.
